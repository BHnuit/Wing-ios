隐私政策

最后更新时间：2026年2月21日

欢迎使用 Wing。我们深知隐私对您的重要性，因此我们将“隐私优先”作为本应用的核心设计理念。本隐私政策旨在说明 Wing 如何处理您的数据。

1. 数据存储与 100% 本地化

您的所有日记内容、图片、语音记录及设置均完全存储在您的设备本地（通过 iOS 的安全数据库）。我们没有任何后端服务器存储您的日记数据，开发者也无法访问、查看或收集您的任何私人记录。

2. 密钥安全 (Keychain)

Wing 允许您使用第三方 AI 模型进行日记合成。您提供的所有 API Key 均会通过苹果系统最高等级的 Keychain（钥匙串）进行加密安全存储。该信息绝不会被上传至我们的服务器或与其他第三方共享。

3. 第三方 AI 模型服务

当您使用“日记合成”等 AI 功能时，必要的日记片段将会通过加密网络临时发送至您选择的第三方大语言模型提供商（如：OpenAI, Gemini, DeepSeek）。

  • 您的数据传输仅发生于您的设备与目标提供商之间。
  • 请注意，当数据发送至第三方 API 时，其数据处理将遵循各个服务提供商自身的隐私政策。
  • 我们建议您在使用对应的模型前，详细阅读其官方的隐私协议声明。

4. 数据收集与追踪分析

Wing 不含有任何隐藏的追踪脚本、广告 SDK 或用户行为分析工具。我们不会收集您的使用习惯以用于商业或广告目的。您在使用应用时的所有操作均是私密且完全由您自己掌控的。

5. 您的权利

由于所有数据均存储在您的本地设备上：

  • 修改与删除：您可以随时在应用内修改或彻底删除任何日记内容。这会从您的设备上不可逆地抹去对应数据。
  • 数据导出：您可以通过应用内的“数据管理”功能将所有数据以标准格式导出和备份，以便您随时随地拥有您的数据所有权。

6. 本隐私政策的更新

我们可能会适时修改本隐私政策，相关更新将在此页面公布。若有重大变更，我们会在应用内通知您。
